Le but de ce chapitre est de se familiariser avec plusieurs formes vectorielles simples et de les dessiners sur un canva

Dans ce quatrième exercice nous allons nous servir de la superposition pour créer un disque en "arc-en-ciel". Le script javaScript possède deux tableau contenant le rayon et la coueur de plusieurs disques.
Il vous est demandé de faire le travail suivant :

  1. Faire un disque "arc-en-ciel" où chaque couleur est comprise entre deux rayons de disque (ne pas utiliser les outils de d3 mais une boucle for)
  2. D'après vous, quel est l'inconvénient de cette méthode ? (Supposez qu'il n'y ait pas 7 mais des milliers de disques différents)
