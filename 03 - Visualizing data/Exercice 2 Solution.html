Le but de ce chapitre est de comprendre les principes de visualisation à partir d'un tableau de données

Dans ce deuxième exercice, nous allons créé un petit bar chart à partir du tableau de données suivant : [20, 50, 80, 40, 50, 100]. Il vous est demandé de faire le travail suivant :

  1. Dessiner un barchart avec ces données. La largeur des barres est de 20 pixels, la hauteur est proportionnelle aux données mais la valeur maximale doit faire la hauteur totale du canvas
  2. Colorer les barres en bleu
