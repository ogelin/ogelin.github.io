Le but de ce chapitre est de présenter les tree layouts (diagrammes en arbre) permettant de représenter des données en arborescence comme dans un arbre généalogique.

Dans ce premier exercice nous allons introduire les liens entre les noeuds dans les trees layout : on souhaite relier le point du haut aux trois points du bas avec une ligne en forme de S .

Il vous est demandé de faire le travail suivant :

  1. Créer les sigmoides pour relier les points
  2. En utilisant ces sigmoides relier le point du haut aux trois points du bas
