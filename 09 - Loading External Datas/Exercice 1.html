Le but de cet exercice est de charger puis de manipuler un fichier de données externes CSV.

Le format CSV est expliqué à cette page, mais il suffit de retenir qu'il s'agit d'une représentation de données.

Le fichier CSV utilisé dans cet exercice est disponible à cette adresse. Il contient la position, la taille et la couleur des cercles à dessiner

Il vous est demandé de faire le travail suivant :

  1. Charger le fichier CSV avec d3
  2. Dessiner des cercles avec les informations contenues dans le fichier CSV
